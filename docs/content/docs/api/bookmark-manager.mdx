# BookmarkManager API Reference

`BookmarkManager` coordinates bookmark data, UI messaging, and IINA integration.

## Class Overview

```typescript
class BookmarkManager {
  private bookmarks: BookmarkData[] = [];
  private collections: BookmarkCollection[] = [];
  private smartCollections: SmartCollection[] = [];

  private persistence: BookmarkPersistence;
  private importExport: BookmarkImportExport;
  private thumbnailGenerator: ThumbnailGenerator;

  constructor(dependencies: IINARuntimeDependencies);
}
```

<Callout type="info">
  The current runtime does not include a cloud sync handler in <code>BookmarkManager</code>.
</Callout>

## Constructor Behavior

On construction, the manager:

1. Initializes persistence/import-export/thumbnail modules.
2. Loads bookmarks, collections, and resume positions.
3. Registers IINA event listeners and plugin menu items.
4. Defers UI file loading until `iina.window-loaded` when needed.

## Message Protocol (Practical)

`BookmarkManager` handles UI messages such as:

- CRUD + seek: `ADD_BOOKMARK`, `UPDATE_BOOKMARK`, `DELETE_BOOKMARK`, `JUMP_TO_BOOKMARK`
- import/export: `IMPORT_BOOKMARKS`, `EXPORT_BOOKMARKS`
- reconciliation: `RECONCILE_FILES`, `FILE_RECONCILIATION_REQUEST`
- collections and batch operations
- timeline/loop/navigation actions

And emits updates such as:

- `BOOKMARKS_UPDATED`, `BOOKMARK_DEFAULTS`, `CURRENT_FILE_PATH`
- `BOOKMARK_ADDED`, `BOOKMARK_DELETED`, `BOOKMARK_JUMPED`
- `IMPORT_RESULT`, `EXPORT_RESULT`, `PLAYBACK_STATUS`, `ERROR`
