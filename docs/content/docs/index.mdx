# Welcome to IINA Plugin Bookmarks

**IINA Plugin Bookmarks** is a comprehensive bookmark management system designed specifically for the IINA media player. It provides intelligent media title detection, sophisticated bookmark organization, and seamless data management capabilities.

## Features

<Cards>
  <Card title="🎬 Smart Media Detection" href="/docs/features/media-detection">
    Automatic media title detection with intelligent fallback to cleaned filenames
  </Card>
  <Card title="📚 Bookmark Management" href="/docs/features/bookmark-management">
    Create, organize, and manage bookmarks with timestamps and descriptions
  </Card>
  <Card title="🏷️ Tag System" href="/docs/features/tags">
    Flexible tagging system for categorizing and organizing your bookmarks
  </Card>
  <Card title="📤 Export & Import" href="/docs/features/data-management">
    Export bookmarks to JSON/CSV and import from various formats
  </Card>
</Cards>

## Quick Start

<Steps>
  <Step>
    ### Installation
    
    Install the plugin in your IINA plugins directory:
    ```bash
    # Download and extract to IINA plugins folder
    ~/.config/iina/plugins/
    ```
  </Step>
  
  <Step>
    ### Configuration
    
    Configure the plugin settings through IINA's preferences panel or the standalone window.
  </Step>
  
  <Step>
    ### Start Bookmarking
    
    Begin creating bookmarks while watching your favorite media content!
  </Step>
</Steps>
