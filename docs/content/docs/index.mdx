# IINA Plugin Bookmarks

<Callout type="info" title="Latest Release">
  **Version 1.0.0** - Production-ready bookmark management with advanced filtering, batch operations, and performance optimizations.
</Callout>

## ðŸš€ Quick Start

<Steps>
  <Step>
    ### Installation & Setup
    
    <Tabs items={['macOS', 'Manual Install']}>
      <Tab value="macOS">
        ```bash
        # Download the latest release
        curl -L -o bookmarks.iinaplgz https://github.com/wyattowalsh/iina-plugin-bookmarks/releases/latest/download/iina-plugin-bookmarks.iinaplgz
        
        # Install in IINA
        open bookmarks.iinaplgz
        ```
      </Tab>
      <Tab value="Manual Install">
        ```bash
        # Extract to IINA plugins directory
        mkdir -p ~/.config/iina/plugins/
        unzip iina-plugin-bookmarks.iinaplgz -d ~/.config/iina/plugins/bookmarks/
        ```
      </Tab>
    </Tabs>
    
    <Callout type="tip">
      The plugin automatically activates when IINA restarts. Look for the "Bookmarks" tab in the sidebar!
    </Callout>
  </Step>

## ðŸ”§ Plugin Architecture Overview

```typescript
// Core plugin structure
class BookmarkManager {
  // High-performance bookmark operations
  async addBookmark(title?: string, timestamp?: number): Promise<void>
  removeBookmark(id: string): void
  updateBookmark(id: string, data: Partial<BookmarkData>): void
  
  // Advanced data management
  async exportBookmarks(options: ExportOptions): Promise<ExportResult>
  async importBookmarks(data: BookmarkData[], options: ImportOptions): Promise<ImportResult>
  
  // Search and filtering
  getBookmarks(filePath?: string): BookmarkData[]
  searchBookmarks(query: string): BookmarkData[]
  
  // Performance optimizations
  private cache: BookmarkCache
  private debouncedSave: (...args: any[]) => void
  private memoizedGetBookmarks: (filePath?: string) => BookmarkData[]
}
```

<Callout type="info" title="Performance Features">
  The plugin includes enterprise-grade performance optimizations:
  - **Indexed caching** for O(1) bookmark lookups
  - **Debounced operations** to prevent excessive I/O
  - **Memoized functions** for frequently accessed data
  - **Batch processing** for large import/export operations
</Callout>
