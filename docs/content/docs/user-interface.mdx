## Sidebar

The sidebar is the default management surface.

### What it supports

- Add bookmarks
- Import/export bookmarks
- Check moved files (reconciliation)
- Filter/search bookmarks
- Toggle scope: **This File** vs **All Files**
- Jump, edit, and delete bookmarks

### How to access

- **View → Toggle Sidebar** (or `Cmd+Shift+B` in IINA)
- Open any media file and select the **Bookmarks** tab

<Callout type="tip">
  For quick capture without opening dialogs, use <strong>Ctrl+B</strong> (Quick Bookmark).
</Callout>

## Overlay

The overlay is optimized for quick navigation while watching video.

### What it supports

- Search/filter visible bookmarks
- Jump to selected bookmark
- Close with `Esc`

### How to access

- **Plugin menu → Toggle Bookmarks Overlay**
- Optional: bind your own hotkey in IINA plugin shortcut settings

## Standalone Window

The standalone window provides a larger workspace for heavy editing sessions.

### What it supports

- Full bookmark list management
- Advanced filtering and search
- Batch actions and timeline tools
- Import/export and reconciliation actions

### How to access

- **Plugin menu → Manage Bookmarks**

## Plugin Menu Actions

The plugin registers these menu actions:

- **Add Bookmark at Current Time**
- **Manage Bookmarks**
- **Toggle Bookmarks Overlay**
- **Quick Bookmark** (`Ctrl+B`)
- **Next Bookmark** (`Ctrl+]`)
- **Previous Bookmark** (`Ctrl+[`)
