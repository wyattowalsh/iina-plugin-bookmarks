# Cloud Sync (Planned)

<Callout type="warning">
  Cloud sync providers are <strong>not shipped</strong> in current plugin releases.
</Callout>

## What ships today

Use export/import to back up or move bookmarks between machines:

- Export bookmarks from **Manage Bookmarks**
- Transfer the exported JSON/CSV file
- Import on another machine from **Manage Bookmarks**

## Recommended backup and transfer flow

<Steps>
<Step>

### Export from your source machine

1. Open **Plugin menu → Manage Bookmarks**
2. Click **Export Bookmarks**
3. Save as **JSON** (recommended) or CSV

</Step>

<Step>

### Transfer the export file

Move the exported file using any method you trust (AirDrop, cloud drive app, USB, etc.).

</Step>

<Step>

### Import on the destination machine

1. Open **Plugin menu → Manage Bookmarks**
2. Click **Import Bookmarks**
3. Choose duplicate handling (skip/replace/merge) and confirm

</Step>
</Steps>

<Callout type="info">
  Detailed import/export behavior is documented in [Data
  Management](/docs/features/data-management).
</Callout>

## Roadmap status

Cloud provider integrations remain on the roadmap.
No release date or provider commitments are currently published.
